# Melon Contest Blueprint

## How to run?

[x] Open this folder on VSCode.
[x] Click on `View` -> `Terminal`.
[x] Run `npm i`.
[x] Start `mongodb`.
[x] Change the `COOKIE_SECRET` on the `.env` file.
[x] Run `npm run dev:server`.
[-] Open another terminal and run `npm run dev:assets`.

## How to deploy?

- Run `npm run build`.
- Run `npm start`.
